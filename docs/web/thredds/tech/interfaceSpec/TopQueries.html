Common Queries for a Data Service such as TDS

Catalog Search

  1. for a specified time interval, get list of models from which output is available

Model Data

  1. for a specified model, get what parameters are available
  2. for a specified model, get what times (interval, resolution) are available
  3. for a specified model, get what spatial region and resolution is available
  4. for a specified model, spatial region, time interval, and parameter, get gridded data

Station Data

  1. for a specified time interval, get what stations are available
  2. for a specified station, get what times are available
  3. for a specified region and time interval, get what stations are available
  4. for a specified station, get what parameters are available
  5. for a specified station, time interval, and parameters, get observational data
