CrawlableDataset

last updated: 3 August 2006

A CrawlableDataset represents an abstract dataset that is part of a  hierarchical dataset collection. Parent and child datasets can be accessed allowing the collection to be crawled.The CrawlableDataset interface is a generalization (and simplification) of the java.io.File class but can be used to represent any hierarchical collection.

Issues and Future Development

  • A number of CrawlableDataset methods throw IOExceptions. Not sure IOException will always make sense, convert to CrDsException?
  • CrawlableDatasetFactory.createCrawlableDataset() throws a bunch of implementation specific exceptions. Change to exceptions more appropriate to the abstraction.
  • The factory always removes trailing slashes ("/"). In current OPeNDAP, a trailing slash means it is a collection (dods_dir) request.
