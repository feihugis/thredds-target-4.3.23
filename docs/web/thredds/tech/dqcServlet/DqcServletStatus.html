THREDDS DQC Servlet - Status

Comments to THREDDS mail list

The THREDDS DQC Servlet is a framework for providing DQC services. These DQC services include providing DQC documents and accepting and responding to the query requests described by the DQC documents.

Current Stable Release

THREDDS DQC Servlet version 0.3 (released 5 August 2004).

This software is currently an alpha release and uses THREDDS DQC Spec 0.3.

To download, go to the installation page.
