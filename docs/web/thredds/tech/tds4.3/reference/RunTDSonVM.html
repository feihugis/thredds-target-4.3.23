Running the Thredds Server on a Virtual Machine

Unidata staff have created a "Unidata on a memory stick" downloadable virtual machine (VM) containing a working installation of Tomcat, the TDS, the LDM, and other Unidata tools. An image of the VM is available for download here. The VM itself runs a 32-bit Community Enterprise Operating System (CentOS), and is smaller than 4 gigabytes in size. In order to "play" them VM, you will need a VM player such as:

  • VMWare
  • Some other VM player
Playing the VM is as easy as firing up the VM player and "running" VM, as is.
