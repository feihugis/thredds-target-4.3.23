Building TDS from Source Code

TDS Source Code Build

TDS source is now hosted on GitHub, and the source jar is no longer available, and ant builds are no longer supported. To build version 4.3 from source with maven:

git clone git://github.com/Unidata/thredds.git thredds
cd thredds
mvn install

The thredds.war artifact will be found under thredds/tds/target.
