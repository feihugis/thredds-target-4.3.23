Dataset Query Capabilities - Status

Comments to THREDDS mail list

THREDDS Dataset Query Capabilities consists of two parts: 1) documents that are intended to describe the set of available queries for a given dataset/collection; and 2) a service that handles (returns a response to) those queries. The response is assumed to be an InvCatalog unless otherwise specified in the DQC document.

DQC Service

For each dataset that is described with a DQC document, a service is needed to respond to the queries allowed. The response is given in the form of a THREDDS InvCatalog document that contains the datasets that match the query.
