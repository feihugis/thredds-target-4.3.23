CatalogGen: Scheduled Catalog Generation

The CatalogGen software was originally written for command-line generation of THREDDS catalogs. It was generally used to schedule catalog generation for dynamic data collection or for one-off catalog generation. The CatalogGen servlet was written to provide a simple interface for scheduling catalog generation.

Current Status

The CatalogGen software is no longer actively developed (and is only minimally supported). In particular, the old style configuration format (pre-datasetScan) that it uses is no longer being developed. We would like to convert the CatalogGen functionality over to the datasetScan configuration format. However, with our current resources this is not very high on our list.
