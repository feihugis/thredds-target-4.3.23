Viewer Tab

Show data values

Select a variable in the right table, right click for context menu and choose "NcDump Data". That takes you to the "NcDump Variable Data" popup window. The full extent of the variable's indexes are shown in F90 form (start:end:stride), where start and stop are inclusive and 0 based. For example:

varName(0:105483:1, 0:0:1, 0:0:1, 0:0:1)

You may now edit that spec to dump just the values you want. Its a good idea to make the number of values reasonably small if possible.
