Changes to NcML

New features in version 4.0:

netcdf Element

The enhance attribute of the netcdf element is now a String, which can be set to All, ScaleMissing, or CoordSystems. For backwards compatibility, a value of true means All. See NetcdfDataset.EnhanceMode.

Ths iosp, iospParam and buffer_size attributes of the netcdf element allow one to specify which IOServiceProvider to use, to pass a arbitrary string to the IOServiceProvider, and to set the size of the buffer that is used when reading the file.

The section attribute of the netcdf element specifies the tile section.

aggregation element

promoteGlobalAttribute element, cachValue element

timeUnitsChange attribute can now be applied to any outer aggregation
